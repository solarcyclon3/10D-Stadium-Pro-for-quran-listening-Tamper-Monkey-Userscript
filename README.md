# 10D-Stadium-Pro-for-quran-listening-Tamper-Monkey-Userscript
10D Stadium Pro for quran listening on youtube tamper monkey

For those who like echoes or stadium-like sound,
this type sound is more soothing for the ears and for long-time listening (especially of Sheikh Shuraim, Sheikh Sudais or any other reciter), but unfortunately we don't find every videos or audios with this kind of sound.

and Editing this kind sound effect for every audio is a nightmare. So came up with a solution, a Tampermonkey script that lets you customize the sound.

settings:

 - Distance from the speaker (simulates air absorption & volume loss)

- Stadium Echo Limits

- If u like a rotating effect like 10D (and an Ear Pressure setting in case of rotating)

U can use it via Tampermonkey for Edge, Firefox, Chrome, or Chromium-based browsers. In case of android phone, the Edge browser lets you install Tampermonkey. Just create a new script and replace the entire script with it.
